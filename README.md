# Claude Code Agency Skills

> Skills e templates especializados para agências de IA usarem Claude Code na entrega de projetos de forma mais rápida e eficiente.

**Preço:** €39-79 | **Status:** Em desenvolvimento | **Lançamento:** Em breve

---

## 📦 O que está incluído

- **Skills especializadas** para Claude Code
- **Templates de projetos** (Next.js, SaaS, automações)
- **Playbook de execução** com Claude Code
- **Prompts otimizados** para diferentes casos de uso
- **Workshops e tutoriais** em vídeo (bônus)

## 🎯 Público-alvo

Agências de IA e desenvolvedores que querem:
- Entregar projetos 10x mais rápido
- Padronizar qualidade de entrega
