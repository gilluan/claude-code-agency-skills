# Claude Code Agency Skills - Resumo do Projeto

## Visão Geral

Produto digital ensinando agências de IA a usarem Claude Code como ferramenta principal de entrega, aumentando produtividade e padronizando qualidade.

## Objetivos

1. Criar skills especializadas para Claude Code
2. Desenvolver templates prontos de projetos
3. Criar playbook de execução passo a passo
4. Produzir conteúdo em múltiplos formatos

- [ ] 10+ skills criadas
