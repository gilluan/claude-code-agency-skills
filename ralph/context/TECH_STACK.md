# Tech Stack - Claude Code Agency Skills

## Produto Digital

**Formato principal:**
- PDF (ebook/playbook)
- Templates (repositório GitHub)
- Prompts (arquivos markdown)
- Vídeos (bônus - Loom/YouTube)

**Ferramentas de produção:**
- Escrita: Obsidian / Markdown
- Design: Canva / Figma
- Code: VS Code + Claude Code
- Versionamento: Git + GitHub

## Página de Vendas

**Stack:**
- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Deploy: Cloudflare Pages
- Analytics: Plausible / Google Analytics
- Checkout: Stripe

## Estrutura de Arquivos

```
Product/
├── ebook/              # PDF principal
├── skills/             # Skills para Claude Code
├── templates/          # Templates de projetos
├── prompts/            # Prompts otimizados
└── videos/             # Conteúdo em vídeo (bônus)
```
